# 📚 Smart Library Assistant

## Description
A smart library management system with AI-powered book recommendations using content-based filtering.

## Features
- 🔍 Search books by title, author, or genre
- 🌟 AI-based book recommendations using TF-IDF and cosine similarity
- 📥 Borrow and return books
- 📊 Library statistics and analytics
- 🎯 Genre-based recommendations
- 🔗 Similar book suggestions
- ⭐ High-rated book recommendations
- 📖 Book details view
